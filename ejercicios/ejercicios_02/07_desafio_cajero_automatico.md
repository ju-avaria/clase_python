# 🏧 Desafío 1 — Cajero Automático

## 🎯 Objetivo

Practicar:

- Variables
- Condicionales
- Operaciones matemáticas
- Entrada de datos

---

## 📌 Instrucciones

Crear un programa que simule un cajero automático.

El programa debe:

1. Tener un saldo inicial guardado en una variable
2. Solicitar al usuario cuánto dinero desea retirar
3. Verificar:
   - Si tiene saldo suficiente
   - Si el monto es válido
4. Mostrar:
   - Nuevo saldo
   - Mensaje de error si no tiene fondos

---

## 💡 Ejemplo esperado

```text
Saldo disponible: 50000

Ingrese monto a retirar: 10000

Retiro realizado correctamente
Nuevo saldo: 40000
```

---

## ⚠️ Casos a considerar

- No permitir retirar más dinero del disponible
- No permitir números negativos

---

## ✅ Temas aplicados

- Variables
- Condicionales
- Comparaciones
- Operaciones matemáticas
