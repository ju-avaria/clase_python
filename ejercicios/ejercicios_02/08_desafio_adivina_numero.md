# 🎲 Desafío 2 — Adivina el Número

## 🎯 Objetivo

Practicar:

- Bucles
- Condicionales
- Comparaciones
- Variables

---

## 📌 Instrucciones

Crear un programa donde:

1. El programa tenga un número secreto
2. El usuario deba adivinarlo
3. Mientras no adivine:
   - seguir preguntando
4. Mostrar pistas:
   - “El número es mayor”
   - “El número es menor”

---

## 💡 Ejemplo esperado

```text
Adivina el número:

Ingrese un número: 3
El número es mayor

Ingrese un número: 8
El número es menor

Ingrese un número: 5

¡Correcto!
```

---

## ⭐ Extra

Intentar contar cuántos intentos realizó el usuario.

---

## ✅ Temas aplicados

- While
- If
- Variables
- Comparaciones
